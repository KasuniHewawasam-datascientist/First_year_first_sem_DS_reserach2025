# First_year_first_sem_DS_reserach2025
Data Science analysis of survey responses on screen-based communication

# Impact of Screen-Based Communication on Society 

## Project Overview 
This project analyzes primary survey data collected via Google Forms to explore the societal, 
social, and health-related impacts of screen-based communication.

## Dataset 
- Source: Self-collected Google Form responses
- Number of participants: 38
- Format: CSV -
- Location: Sri Lanka
   
## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
  
## Repository Structure 
- data/: Contains the survey dataset
- notebook/: Contains the Jupyter Notebook analysis
- README.md: Project documentation
- data_dictionary.md: Description of dataset variables
- metadata.md: Dataset-level information
   
## How to Run the Analysis 
1. Download or clone the repository 
2. Open the Jupyter Notebook in the `notebook` folder 
3. Run all cells sequentially
