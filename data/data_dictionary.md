# Data Dictionary 
| Column Name | Description | Data Type | 
|------------|------------|----------| 
| screen_experience | Whether the participant has experience using digital screens | Categorical |
| tech_makes_com_easier | Agreement level on technology making communication easier | Likert scale | 
| continue_screen_com | Preference to continue screen-based communication | Categorical | 
| reasons_screen_com | Reasons for choosing screen-based communication | Multi-select | 
| difficulty_text_expression | Difficulty expressing emotions via text | Categorical | 
| social_awkwardness | Feeling awkward during face-to-face interaction | Categorical | 
| face_to_face_difficulties | Difficulties in face-to-face communication | Multi-select | 
| social_skill_impact | Impact of screen communication on social skills | Multi-select | 
| wear_spectacles | Whether participant wears spectacles | Binary | 
| doctor_visit | Doctor visits due to screen-related issues | Binary | 
| noticed_symptoms | Noticing physical or mental symptoms | Binary | 
| symptoms_types | Types of symptoms experienced | Multi-select | 
| need_physical_socializing | Need for physical social interaction | Categorical | 
| preferred_com_mode | Preferred communication mode | Categorical | 
| reason_for_preference | Reason for communication preference | Text |
